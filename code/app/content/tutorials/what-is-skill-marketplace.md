## The Idea

Skills Marketplace is a self-hosted place for teams to publish, discover, install, update, and improve AI agent Skills.

A Skill can contain:

- A compact `SKILL.md` instruction file.
- Longer references, templates, or examples.
- Scripts and assets that help an agent perform repeatable work.

The Web app is for humans. The CLI is for agents and local setup. Together they let a team package knowledge once and reuse it across Codex, Claude Code, Cursor, Antigravity, or another compatible agent.

![Marketplace home](/tutorials/screenshots/marketplace-home.png)

## The Loop

1. A creator publishes a Skill.
2. A user or agent searches the marketplace.
3. The Skill is installed into a local agent directory.
4. The agent uses the Skill in the right context.
5. When something fails, the user or agent sends feedback.
6. The creator improves the Skill and publishes a new version.

This is not just a prompt gallery. It is a lightweight knowledge distribution and feedback loop for agent work.
