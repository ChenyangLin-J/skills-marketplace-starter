场景 3 / 使用失败后把反馈带回 creator
User
这个报告 Skill 没有问受众，直接开始写了。

Agent
这是一个可改进的 Skill 行为。我会把复现信息反馈给作者，方便下一版补充“先确认受众”的规则。

Bash(agent-skills feedback @demo/report-writer \
  --type issue \
  --agent codex \
  --message "Report writer should ask for audience before drafting" \
  --context "User asked for a project report. The Skill started drafting without confirming audience or purpose.")

反馈已提交。creator 可以在 Web 详情页或「My Skills」里看到这条真实失败样例，并据此更新 Skill。
